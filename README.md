# FastRbx

FastRbx is a Shortcut for iOS, iPadOS, and macOS that works as a custom
launcher for Roblox using deep links.

## Installation

1.  Open the Releases section.\
2.  Download the Shortcut file.\
3.  Open it in the Shortcuts app.\
4.  Confirm the installation.

## Features

"Open Place" --- open a Place by ID.\
"Join to Player" --- join a specific player.\
"Join to JobId" --- join a specific server.\
"Open Place with LaunchData" --- launch with custom data.\
"Change Avatar Background".\
"FastFlags".\
"Tab Glitch".

It combines multiple launch methods and client options into a single
Shortcut.

FastRbx is not affiliated with Roblox. Some features may depend on the
current client version.
